# **App Name**: TaskFlow

## Core Features:

- User Authentication: Secure login/authentication using email and password with different user roles.
- Dashboard Overview: Display a summary of tasks by status and priority using charts and recent activity.
- Task Management: Create, view, edit, and delete tasks with detailed information and file upload capabilities.
- Financial Tracking: Track initial demand, official payment, and motivation amounts associated with tasks.
- Document Management: Upload, view, download, and print documents related to tasks.
- Settings Configuration: Allows administrators to configure task-related fields (e.g., labels, locations, priority) via a dropdown list. Includes user creation with various roles and rights.
- Generate smart labels.: Suggest labels for new tasks based on the task title and description using a generative AI tool.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to promote trust and productivity.
- Background color: Light gray (#F0F4F7) for a clean, professional look.
- Accent color: Dark blue (#3F51B5) to highlight important actions and data.
- Body font: 'PT Sans', sans-serif, for clear, readable text in tables and descriptions.
- Headline font: 'Space Grotesk', sans-serif, for headings and titles.
- Use clean, modern icons from a consistent icon set (e.g., Material Design Icons) to represent task statuses, priorities, and actions.
- Implement a responsive layout with clear sections for dashboards, tasks, financials, and documents. Use cards and tables to organize information.
